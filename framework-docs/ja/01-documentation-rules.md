# ドキュメント言語ルール

このリポジトリでは、多言語ドキュメントに関して以下のルールに従います。

## 使用言語

日本語を正本（source of truth）とします。
英語ドキュメントは、日本語ドキュメントの翻訳です。

## 構成

日本語のドキュメントは docs/ja/ 配下に配置します。
英語のドキュメントは docs/en/ 配下に配置します。
docs配下のドキュメントファイルは、頭に数字2桁のドキュメント番号をつけて、 "00-[ドキュメント名].md" という形式にします。
ファイル名は言語間で 必ず同一 にします。
docs配下以外のテンプレートやサンプルファイルの日本語版は、[ファイル名].ja.xxx という形式で配置します。
例）./CLAUDE.ja.md.template

## ファイル命名規則

"00-[ドキュメント名].md"の[ドキュメント名]部分は、小文字の kebab-case を使用します。
ファイル名は英語のみを使用します。

## README

README.ja.md は日本語版（正本）です。
README.md は英語版です。

両者は必ず相互にリンクさせます。

翻訳ポリシー
英語翻訳が存在しない、または内容が古くなっている場合は、その旨を明確に記載しなければなりません。
