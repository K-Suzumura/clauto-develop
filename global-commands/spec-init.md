# /spec:init - 仕様書テンプレート生成

`docs/spec.md` に仕様書テンプレートを生成してください。

## 実行内容

1. `docs/` ディレクトリが存在しない場合は作成
2. 以下のテンプレートで `docs/spec.md` を生成
3. 既存ファイルがある場合は上書き確認
